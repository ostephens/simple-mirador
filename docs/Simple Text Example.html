Simple Mirador - The Text Example

  1. Examples

This extension uses the IIIF image viewer Mirador to produce a customised presentation of a bespoke selection of IIIF manifests[1]. It is an example of a special page that can be added and presented using the simple-site project.

For more information about the simple-site project and details about how this example work please see the simple-site webpages.

Summary

This very simple example page has been created using only a simple text list of IIIF manifests, for more information see here.

The Viewer

  • [1] IIIF manifests are simply a structured list of web based images resources that can be viewed in a IIIF compliant viewer, for more details see https://iiif.io/api/presentation/2.1/#table-of-contents
